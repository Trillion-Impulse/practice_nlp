# NLP란?
- NLP는 컴퓨터가 인간의 언어를 이해하고 처리하도록 만드는 기술
- Natural Language Processing
- ex
    - 사람이 보면 바로 이해하는 문장
        ```
        삼성전자 주가가 오늘 크게 상승했다.
        ```
    - 컴퓨터는 그냥 문자열 데이터로만 봄
        - NLP는 위의 문장을 아래와 같이 바꿈
            ```
            [삼성전자] [주가] [상승]
            ```
        - 문장을 의미 있는 데이터로 변환하는 과정
            - 토큰화, 형태소 분석 등 사용

## NLP는 언제 사용하는가?
- 다양한 텍스트 데이터를 처리하는 경우에 사용
- ex
    - 뉴스
        ```
        삼성전자는 AI 반도체 시장에서 경쟁력을 강화하고 있다.
        ```
    - 쇼핑몰 리뷰
        ```
        배송 빠르고 제품 좋습니다
        ```
    - SNS
        ```
        이 영화 진짜 재밌다
        ```
- 텍스트 데이터는 그대로 분석하기 어렵기 때문에 NLP 처리

## NLP 왜 하는가?
- 여러가지 목적이 존재
    - 키워드 추출
        ```
        삼성전자 AI 반도체 경쟁력 강화
        ↓
        삼성전자
        AI
        반도체
        ```
    - 감성 분석
        ```
        배송 빠르고 제품 좋습니다
        ↓
        긍정
        ```
    - 문서 분류
        ```
        삼성전자 반도체 투자 확대
        ↓
        카테고리: 경제
        ```
    - 요약
        ```
        긴 기사
        ↓
        핵심 문장 요약
        ```

## NLP 처리 단계
- 보통의 순서
    ```
    텍스트 수집
    ↓
    텍스트 정제
    ↓
    토큰화
    ↓
    불용어 제거
    ↓
    벡터화
    ↓
    분석 / 모델
    ```
- 텍스트 수집
- 텍스트 정제
    - 불필요한 문자 제거
        ```
        삼성전자!!! 주가 상승???
        ↓
        삼성전자 주가 상승
        ```
- 토큰화 (Tokenization)
    - 문장을 단어 단위로 나누는 작업
        ```
        삼성전자 주가 상승
        ↓
        ["삼성전자", "주가", "상승"]
        ```
- 불용어 제거
    - 텍스트에서 분석에 크게 영향을 미치지 않는 단어들을 제거하는 과정
        ```
        나는 오늘 학교에 갔다
        ↓
        학교
        갔다

        불용어
        - 나는
        - 오늘
        - 에
        ```
- 벡터화
    - 컴퓨터는 숫자만 이해하므로 단어를 숫자 벡터로 변환
        ```
        삼성전자 → [0.12, 0.88, 0.44 ...]
        ```
    - 여기서 사용하는 기술
        - Word2Vec, BERT, embedding

## Python의 NLP 도구
- 표
    | 도구                        | 특징                    | 장점                       | 단점               |
    | ------------------------- | --------------------- | ------------------------ | ---------------- |
    | KoNLPy                    | 형태소 분석, 명사 추출         | 한국어 특화, 무료               | 속도 느림, 최신 모델 미지원 |
    | NLTK                      | 토큰화, 품사 분석, 불용어 제거    | 학습 자료 많음, 간단한 NLP 가능     | 한국어 지원 미약, 속도 느림 |
    | spaCy                     | 빠른 토큰화, 품사 분석, NER    | 속도 빠르고 최신 모델             | 한국어 모델 제한적       |
    | Hugging Face Transformers | BERT, GPT 등 최신 NLP 모델 | 성능 높음, 다양한 pretrained 모델 | 학습/추론 느림, GPU 필요 |
    | scikit-learn              | TF-IDF, 벡터화, 기본 ML 모델 | 간단한 NLP 분석, 통계/ML 통합 가능  | 고급 딥러닝 모델에는 한계   |
